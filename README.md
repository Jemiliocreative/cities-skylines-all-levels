# cities-skylines-all-levels
In this repository you will find a file thats give your all levels in the game.

# How to install?
Copy the Jemiliomap1.crp file and past it in C:\Users\YOUR_NAME\AppData\Local\Colossal Order\Cities_Skylines\Maps!
Restart/start your game and hit new game. Choose the map Mable Island!
Clik continue!

# What is in the map?
There are 
  - a lot of trees
  -a lot of oil
  -a lot of stone
  -2 incoming and 2 outgoing roads
  -3 train connections
  -3 ship connections
  -4 plane connections
  -1 big sea
  -2 rivers
  -a waterfall!!
  -mountains
